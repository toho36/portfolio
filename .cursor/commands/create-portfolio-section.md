# Create Portfolio Section

Generate a portfolio section component (Hero, About, Projects, Contact) following portfolio best practices.

## What to do:

1. Ask user which section to create:
   - Hero section
   - About section
   - Projects showcase
   - Contact section
   - Custom section
2. Generate section component with:
   - Proper semantic HTML (section, article, etc.)
   - TypeScript interfaces for props
   - Responsive design (mobile-first)
   - Accessibility features (ARIA labels, keyboard nav)
   - SEO considerations (heading hierarchy)
   - Optional animations (scroll-triggered)
3. Include styling suggestions or Tailwind classes
4. Add proper metadata/SEO if needed

Follow all rules in:

- `.cursor/rules/02-react-components.mdc` (React patterns)
- `.cursor/rules/09-portfolio-patterns.mdc` (Portfolio patterns)
- `.cursor/rules/08-performance.mdc` (Performance)

## Section Types:

### Hero Section

- Name, title, brief bio
- Call-to-action buttons
- Hero image with Next.js Image
- Subtle animations

### About Section

- Personal story
- Skills/technologies
- Interests/hobbies
- Timeline or experience

### Projects Section

- Project grid with filtering
- Project cards
- Technology badges
- Links (live, GitHub)

### Contact Section

- Contact form
- Social links
- Availability info
